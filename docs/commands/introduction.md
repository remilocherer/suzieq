Commands are the basis of interacting with data in suzieq. When you use the
suzieq-cli or the GUI, you supply command, verb, and filters. Verbs
and filters apply to the command given.

Current commands are:

 - arpnd
 - bgp
 - device
 - evpnVni
 - interface
 - lldp
 - mac
 - mlag
 - ospf
 - path
 - route
 - sqpoller
 - table
 - vlan
